## 🛠️ Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/NutriAI.git
cd NutriAI
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Start the application
```bash
streamlit run NutriAI_bot.py
```
